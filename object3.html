3. Bootstrap

Bootstrap 5

Bootstrap is the most popular free CSS framework for developing a responsive website. A website is responsive if the appearance changes depending on the screen size. It resizes the content to fit different screen widths. Implementing all this yourself can be time-consuming and challenging. But Bootstrap takes care of that for you. Learn how Bootstrap works, and apply it to your website. You can account for different devices with a single code base and make your website look professional with its built-in templates and themes.

*Note: This course uses Sublime Text Editor, but you do NOT need to download it. You can use Visual Studio Code instead.
